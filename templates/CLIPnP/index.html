About

CLIPnP is an open-source, lightweight Java port-forwarding program. With CLIPnP, you can run as little as one command to set up port forwarding for your network! There's no need to mess with network configuration or router settings, just download CLIPnP!

Software

For developers, our software is entirely documented with JavaDocs, meaning it's quick and easy to understand what's going on in the software, and making any changes or additions a breeze to implement. The Github repository is completely public as well, so developers and users can stay up-to-date with the latest software.

Licensing

Our software is completely open-source and free to use with the GNU Lesser General Public License Version 2.1. Feel free to transform it however you want!

Support

CLIPnP comes built-in with user-support, so just by running the command: 'help' you can easily find what you're looking for, and solve any syntax issues. More information related to user-support can be found on the official Github repository here.
